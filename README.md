## üöÄ **Installation and Usage**

### Method 1: Built-in PHP Server (Recommended)
```bash
php -S localhost:8000
```
Then open http://localhost:8000/propaint.php in your browser.

### Method 2: With Apache/Nginx Server
Place `propaint.php` in your web folder and access it via your browser.

### Method 3: Docker
```bash
docker-compose up -d
```

### Requirements
- PHP 7.0+ with `zip` extension (for font upload)
- Modern browser (Chrome, Firefox, Safari, Edge)
